Boto Querying a HIT:

This page provides a demonstration of how to get information about a HIT. Please note, your AWS Access Key ID and your AWS Secret Access key are like your username and password to your account. This website does not store them, but if you feel uncomfortable inputting them here, then don't use this page.

For some reason, this function only works with relatively recent hits. After a few months hits become unavailable on boto.
